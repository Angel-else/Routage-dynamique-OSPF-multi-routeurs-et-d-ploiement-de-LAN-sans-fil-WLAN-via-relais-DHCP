# Routage-dynamique-OSPF-multi-routeurs-et-d-ploiement-de-LAN-sans-fil-WLAN-via-relais-DHCP
Ce projet consiste en la conception et la configuration d'une infrastructure réseau multi-sites simulée sur Cisco Packet Tracer. L'objectif principal était de mettre en place une communication fluide par connexion sans fil et automatique entre trois segments réseaux distincts tout en centralisant la gestion des adresses IP.

Fonctionalités techniques

Routage Dynamique : Configuration du protocole OSPF entre trois routeurs pour garantir une convergence rapide et une redondance des routes.
Utilisation d'interface loopback.
Gestion IP centralisé avec DHCP : Configuration d'un routeur en tant que serveur DHCP principal gérant 3 pools d'adresses différents.
Relais DHCP : Utilisation de la fonctionnalité ip helper-address par pour permettre l'attribution d'IP aux hôtes situés dans des réseaux distants.
Connectivité Sans Fil (WLAN) : Déploiement de points d'accès (Access Points) pour assurer la connectivité des terminaux mobiles.
